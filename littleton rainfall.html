ABOUT THE DATA
All precipitation data comes from the real KAPA (Centennial Airport) ASOS tipping-bucket gauge via Iowa State's IEM archive — the same physical sensor network as the MHFD ALERT map and CSU precipitation maps. Hourly readings, no model interpolation.

Forecast is the official NWS 7-day for your grid point.

Annual average: 17.5" based on NOAA normals for Littleton/south Denver (not Denver airport which is drier at 14.3").
