ABOUT THE DATA
Precipitation (24h/3d/7d): Real tipping-bucket gauge data from Centennial Airport (KAPA, ~3 mi) via the free NWS API — the same physical gauges visible on the MHFD ALERT map.

Precipitation (30d/1y): NOAA HRRR model estimate via Open-Meteo. Model data can miss localized convective storms — cross-check with the MHFD link below.

Forecast: Official NWS 7-day forecast for your grid point.

Annual average: 17.5" based on NOAA normals for Littleton/south Denver metro (not Denver airport which is drier at 14.3").
